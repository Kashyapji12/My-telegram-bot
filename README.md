# 🔥 CodeFlow - SIGMA Edition 🚬

AI-powered Telegram bot with sigma roasting!

## ✨ Features

✅ **AI Coding Help** - Gemini powered  
✅ **Code Execution** - Python, JavaScript, Bash  
✅ **File Analysis** - Upload and analyze code  
✅ **Sigma Roasting** - Every response has attitude 💀  
✅ **Premium Emojis** - Modern telegram stickers  

## 🎯 Commands

```
/start - Welcome message
/help - Full guide
/ask <question> - Ask AI
/run - Code execution
/sigma_on - Max roasting 🔥
/sigma_off - Normal mode
```

## 🚀 Deploy

```bash
git clone https://github.com/Kashyapji12/my-telegram-bot.git
cd my-telegram-bot
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## 📋 Setup

1. Create `.env` file
2. Add:
```
BOT_TOKEN=your_token
GEMINI_API_KEY=your_key
ADMIN_ID=your_id
```

3. Run: `python main.py`

## 🌐 Deploy on Render

1. Push to GitHub
2. Render.com → New Web Service
3. Select repository
4. Set environment variables
5. Deploy! 🚀
